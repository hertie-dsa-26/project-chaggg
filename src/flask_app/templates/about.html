the project

the task Working in a team of 7, we were tasked to design and develop a comprehensive web application using Flask that performs end-to-end analysis of a dataset.

requirements The application must include exploratory data analysis, interactive visualisations and at least one algorithm implemented from scratch.

next steps We would like to keep working on the following:

  • Before launching on the browser, the app cleans the dataframe, creates subsamples for every crime type and precomputes our 'naive estimator'. This takes some time. We would like to optimise this processes.
  • We found some differences between our from-scratch knn algorithm and the scikit-learn one. Predictions don't match exactly. We would like to tune both algorithms to make them closer.
  • We would like to merge the crime dataframe with other sources of information. Particularly, we would like to add ethnic origin and socioeconomic variables to our controls to find how these variables impact on the probability of an arrest.
